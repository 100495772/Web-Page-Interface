**Diseño de prototipo y desarrollo de un sitio web interactivo y accesible**  
*Tecnologías: HTML, CSS, JavaScript, LocalStorage*  

Implementación de un **prototipo web responsive** garantizando accesibilidad y usabilidad mediante herramientas como **WAVE** y la evaluación **SUS**. 

Este proyecto utiliza Bootstrap 5 para su diseño.
Asegúrate de tener instaladas las librerías:
 https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css
 https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js
